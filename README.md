# LearnBridge  🎓

A web-based platform for managing exams and student grades 
for educational institutions.

## Features
- Role-based authentication (Admin, Teacher, Student)
- Student grade management and transcripts
- Exam scheduling and management
- Absence tracking with justifications
- Automatic average calculation
- PDF report generation

## Built with
- PHP
- MySQL
- HTML5 / CSS3 / JavaScript
- XAMPP / phpMyAdmin

## How to run
1. Clone the repo
2. Import `config/web_base.sql` in phpMyAdmin
3. Configure `config/school.php` with your DB credentials
4. Run with XAMPP
